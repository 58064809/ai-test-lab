# 008｜成熟工具接入评估任务验证

## 任务

请你评估 Schemathesis 是否适合作为当前个人 AI 测试助手的接口测试生成工具。

## 当前项目约束

1. 项目是个人专用 AI 测试助手；
2. 不做团队平台；
3. 不造轮子；
4. 代码实现交给 Codex；
5. 当前已有 Pytest + Allure；
6. 不能写死具体业务；
7. 优先成熟工具、官方生态、工业级方案；
8. 不开放 `shell` / `filesystem_write` / `github_write`；
9. 工具接入应服务真实测试任务，而不是为了堆功能。

## 输出要求

1. 工具用途；
2. 适配当前项目的位置；
3. 接入优先级；
4. 最小接入方式；
5. 不适合解决的问题；
6. 与 Pytest + Allure 的关系；
7. 是否建议现在接入；
8. 给出下一步 Codex 任务清单；
9. 如果当前信息不足，请明确说明需要补充哪些前置信息。
