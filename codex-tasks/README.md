# Codex 任务清单

本目录只存放交给 Codex 执行的任务清单。

项目中的真实任务样例、评分表、模板、Prompt、工具说明等资产应放在对应目录中，不要混在这里。

## 使用原则

1. 这里的文件是“给 Codex 执行的任务”，不是项目文档正文；
2. 任务必须明确输入文件、执行范围、边界约束和验收标准；
3. 代码实现交给 Codex，本目录只写任务清单；
4. 默认直接基于当前分支执行，不要求创建 PR；
5. 不开放 `shell` / `filesystem_write` / `github_write` 这类项目 runtime 能力；
6. 不做团队平台、不做传统测试平台、不写死具体业务；
7. 后续任务按编号递增，例如 `040_xxx.md`、`041_xxx.md`。
